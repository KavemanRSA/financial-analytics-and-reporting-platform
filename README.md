# Financial Analytics & Reporting Platform
## Overview
A budgeting and financial tracking application that Implements transaction categorisation, deposits, withdrawals, fund transfers, balance tracking, and spending analysis features. Features a a financial
visualisation output chart that includes a cost category spending distribution chart (text-based) as part of the app’s reporting logic. This project is ongoing
The project has begun as a lightweight budgeting system and is slated to be progressively expanded into a more feature-rich analytics platform featuring dashboards, automated reporting, and AI-assisted financial
insights.

## Disclaimer
This project is part of my ongoing software engineering and AI development portfolio. It is not being designed for commercial or private use.
